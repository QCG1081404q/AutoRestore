🚀AutoRestore - Windows 系统全自动安装与迁移工具

重装系统后还在逐个下载安装软件？AutoRestore 让你的装机体验从“痛苦数小时”变成“咖啡一杯的时间”！

📖项目简介
AutoRestore 是一款专为 Windows 用户设计的智能系统备份与一键迁移解决方案 。无论是新电脑装机、系统重装，还是工作环境迁移，它都能帮你：

-✅完整备份已安装的软件及安装包
-收集系统中的 exe/msi/msix 等格式安装包
-检测缺失的系统运行库（VC++、.NET、DirectX、Java、Git 等）
-✅一键还原完整的软件环境和目录结构
只需三步→打包→还原，即可在新系统上重建完整的开发/工作环境。

✨核心功能
📦软件安装包备份自动扫描系统中已安装软件的原始安装包（exe/msi/msix/appx等），统一归档至 collected 目录，支持离线安装📋已安装软件清单自动生成软件安装列表，记录每个软件的来源路径和安装️目录结构备份智能扫描并记录 D:/E:盘的完整目录结构，还原后文件位置与原来完全一致🔧运行库自动检测智能检测系统中缺失的运行库（VC++ Redistributable、.NET Framework、DirectX、Java JDK、Git、7-Zip 等），并提示️静默批量安装支持自动生成静默安装参数，批量安装软件，支持自定义安装路径（避免强制安装到 C 盘）🔄一键环境还原提供完整的批处理脚本，在新机器上按顺序自动执行：目录重建→安装包安装→运行库配置💾压缩打包工具使用7-Zip 高级压缩，生成单一压缩包，便于携带和备份🎨WPF 图形界面提供现代化的桌面 GUI 程序（AutoRestore.exe），无需命令行，一键操作

技术栈
类别技术自动化脚本 Batch (.bat)、PowerShell 图形界面 C# WPF (.NET 8) 打包工具7-Zip（高级压缩）安装格式 exe、msi、msix、msixbundle、msu、msp、appx、appxbundle
